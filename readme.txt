== Description ==

*Extends the default Recent posts widget with more options:*

* Display post excerpt when available
* Display thumbnail
* Configure thumbnail size
* Choose the Post Type to display
* Display archive link

[Demo](http://www.osi.lv/)

== Installation ==

1. Upload `recent-posts-widget` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
